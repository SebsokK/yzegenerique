V0.1.0 is Ready !
Before you start, don't forget to set the module : it is made to play any YZE game and customize sheets without any coding knowledge. It should work for "modern" and "classic" YZE games (that means don't try to do Forbidden Lands or Blade Runner with it).
When you launch it, you won't have any setting. You have to check the optionnal rules you want in the module settings.
If you want to play Sleepy Hollow, Horror! RPG or Eldricht Automata, I've got a good news for you : go into the module settings and pick the right preset. It will adapt the whole module with just one click.
Now you're not ready yet... You need to go into the compendiums and import all the tables related to your game (Stress & Critical Injuries).
You're all set : you only have to create your PCs and NPCs (drag and drop the skill and abilities from the compendium).
Have fun !
