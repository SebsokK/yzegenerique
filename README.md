# ⚙️ YZE Generic System — V0.1.0

> **The foundation toolkit for modern Year Zero Engine games on Foundry VTT.**
> Build your own game, adapt existing settings, and customize sheets — **without writing a single line of code.**

---

## 🚀 Welcome to Version 0.1.0

The first public version is here!

This system is designed to let you create and run **modern-style Year Zero Engine games** with maximum flexibility and minimum setup pain.

It supports:

* Custom character sheets
* Modular optional rules
* Drag & drop skills and abilities
* Preset-based game configurations
* No-code customization

---

# 📌 Important Before You Start

This module is meant for **modern** and **classic** YZE implementations.

✅ Works great for games inspired by:

* *Alien RPG*
* *Tales From the Loop*
* *Coriolis*
* *Vaesen*
* *Sleepy Hollow*
* *Horror! RPG*
* *Eldritch Automata*

⚠️ It is **NOT** designed for:

* *Forbidden Lands*
* *Blade Runner RPG*

(Those games use more specialized mechanics and structures.)

---

# 🛠 First Launch Setup

When you first launch the system, **almost nothing is enabled by default**.

This is intentional.

The module is built to be modular, so you can configure only the mechanics you need.

## Step 1 — Configure Optional Rules

Go to:

> **Game Settings → Module Settings**

From there, enable the optional rules you want:

* Stress systems
* Panic mechanics
* Critical injuries
* Push mechanics
* Custom variants
* Etc.

---

# 🎮 One-Click Presets

Want to play one of the included game presets?

Good news.

The system includes ready-to-use configurations for:

* *Sleepy Hollow*
* *Horror! RPG*
* *Eldritch Automata*

## To activate a preset:

Go to:

> **Game Settings → Presets**

Select your game.

✅ The entire module will instantly adapt:

* Colors
* Rules
* UI behavior
* Tables
* Terminology
* Dice behavior

All with a single click.

---

# 📚 Import Required Tables

You're not done yet.

Before playing, open the **Compendiums** and import the tables related to your game.

Usually this includes:

* Stress tables
* Panic tables
* Critical injuries
* Trauma tables
* Custom random tables

Without these tables, some mechanics will not function correctly.

---

# 👤 Creating Characters

Once setup is complete:

1. Create your PCs and NPCs
2. Open the compendiums
3. Drag & drop:

   * Skills
   * Attributes
   * Abilities
   * Talents
   * Gear

onto the sheet.

That's it.
